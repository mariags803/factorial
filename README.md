# Code kata: Factorial


## Description

The factorial function multiplies all the integers and positive numbers 
between the number that arrives by parameter and the number 1.

__Example__

``` js
0   // => 1
1   // => 1
2   // => 2
3   // => 6
8  // => 40320

```

## Run project

__Install__

    npm install

__Run test__

    npm test
